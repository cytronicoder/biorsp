# BioRSP Overview

BioRSP in this repository is a heart-only case study pipeline for directional enrichment on 2D embeddings.

## What This Case Study Is

- A method-validation workflow for known heart biology marker panels and QC confounds.
- A representation-conditional analysis: directional angles are interpreted in the embedding coordinate frame only.
- A donor-aware inferential workflow when donor labels are present.

## What This Case Study Is Not

- A subtype-discovery workflow.
- A claim of embedding-invariant biological direction.
- A causal trajectory model.
