# biorsp
BioRSP is a geometry-aware framework for statistically quantifying directional radial enrichment of foreground signals in 2D omics embeddings using calibrated radar scanning plots and permutation-based inference.

## Installation

Install runtime dependencies:

```bash
pip install -r requirements.txt
```

For development (tests, linters, optional extras):

```bash
pip install -r requirements-dev.txt
```

Or using project extras (if installing via pip/pyproject):

```bash
pip install .[dev]
```
