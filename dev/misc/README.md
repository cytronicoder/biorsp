# BioRSP Scripts

Utility scripts for analysis, visualization, and debugging.

## Visualization

**Note:** The KPMP embedding plotting script has been moved to `case_studies/kidney/plot_kpmp_embedding.py` to be co-located with the kidney case study data and analyses.

## Debugging and Development

### debug_end_to_end.py
End-to-end workflow testing with synthetic data.

### make_end_to_end_workflow.py
Generate workflow diagrams and documentation.

### benchmark_angle_window.py
Benchmark different angular window sizes for sensitivity analysis.

### plot_simulation_csv.py
Plot results from simulation studies.

## Schematic Generation

### make_schematic_diagram.py
Generate method schematic diagrams for papers/presentations.

### make_polar_embedding_figure.py
Create polar coordinate visualizations of embedding structure.
